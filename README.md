
# My Online Store

A simple HTML project created as part of an HTML Basics Course.

## 📌 Project Description

This project demonstrates fundamental HTML concepts by building a basic online store webpage. The page includes:

- A website header
- A product table displaying:
  - Product names
  - Product images
  - Prices
  - Product links
- A customer request form

## 🛠️ Technologies Used

- HTML5
- Inline CSS

## 📚 HTML Concepts Practiced

- Document structure (`<!DOCTYPE html>`)
- Headings and paragraphs
- Tables
- Images
- Hyperlinks
- Forms
- Input fields
- Labels
- Basic styling with inline CSS

## 📂 Project Structure

```
project-folder/
│
├── index.html
└── README.md
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Explore the online store interface.

## 🎯 Learning Objectives

By completing this exercise, I practiced:

- Creating structured web pages using HTML.
- Displaying data in tables.
- Adding images and links.
- Creating forms for user input.
- Applying basic page styling.

## 📸 Features

- Responsive product display using HTML tables.
- Product images from online sources.
- External links to product websites.
- User information form with validation.
